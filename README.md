## Uploading to GoDaddy

### Option 1: File Manager (Easiest)

1. Log into GoDaddy → My Products
2. Find your website hosting → Manage
3. Open **File Manager** or **cPanel**
4. Navigate to `public_html` folder
5. **Delete** existing files (backup first if needed)
6. **Upload** `index.html`
7. Visit dialectforge.com to see changes

### Option 2: FTP Upload

1. Get FTP credentials from GoDaddy hosting settings
2. Use FileZilla or similar FTP client
3. Connect to your server
4. Upload `index.html` to `public_html` folder

### Option 3: Replace Website Builder

If using GoDaddy Website Builder:
1. You may need to switch to hosting-only plan
2. Or use a custom HTML section in the builder
3. Contact GoDaddy support for migration help
